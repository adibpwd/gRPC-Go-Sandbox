# gRPC-Go-Sandbox

### Run Service User
```sh
    cd services/service-user/
    go run main.go
```

### Run Service Post
```sh
    cd services/service-post/
    go run main.go
```

### Run Service Garage
```sh
    cd services/service-garage/
    go run main.go
```


### Run Client
```sh
    cd client/
    go run main.go
```
